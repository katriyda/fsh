# fsh - 极简文件流转枢纽 (Ultra-Minimal File Streaming Hub)

fsh 是一个专注于**“极简主义、高性能、零抽象”**的单体轻量级跨平台文件流转中心。它的目标是提供一个极致纯净的文件上传、分享和管理环境。没有任何沉重的企业级框架抽象、没有多余的依赖项，纯粹依赖 Java 21+ 虚拟线程的强大并发能力，直接且高效地完成文件流转。同时，前端管理控制台亦无任何框架依赖，纯粹通过原生 HTML + CSS + JS 构建。

它非常适合用于个人的网盘平替、内网临时文件交换、跨设备传递文件等轻量级场景。

## 🎯 核心设计哲学

- **无框架依赖前端**: 仅单个 HTML 文件构成整个前端界面，原生 JS/CSS，无 Node.js、无 Webpack/Vite，告别构建烦恼。
- **现代化极优性能**: 核心采用了 **Java 21 Virtual Threads (Project Loom)**。抛弃了传统线程池模型的并发瓶颈，极大幅度地降低了高并发 IO 操作下的内存消耗和上下文切换开销。
- **Golang 风格的 Java**: 以结构体（`Java Record`）为数据层、纯粹的明确过程控制流为逻辑层，坚决屏弃 `AOP（切面）` 等影响代码阅读直观性的魔法操作。没有注解满天飞，逻辑所见即所得。
- **极简部署与存储**: **无数据库** 依赖！应用启动时，所有元数据会自动从存储在本地文件系统各个文件身边的 `.json` 描述文件中重建。真正的绿色运行。

## ✨ 核心特性

- ⚡ **无感上传进度**: 支持拖拽文件，无论文件大小，流传输自动落盘，全程原生事件进度条反馈。
- 🔗 **纯净短链接**: 通过 Base62 算法为每个上传的文件生成极其简短的 URL。下载链接保持整洁，无文件名污染，而在真下载时又能够依靠 `Content-Disposition` 完美还原原始名称。
- 🛡️ **安全隔离与删除**: 拥有可配置的管理后台专属路径（Admin Prefix），支持在线删除流转记录，并具备严格安全校验，防止任意文件删除和路径遍历攻击。
- 📦 **免数据库持久化**: 不使用笨重的关系型数据库，所有数据元采用与正文绑定的 JSON 文件伴随存储。搬家、迁移如同搬迁普通文件夹一般容易，随时随地在任何主机上恢复服务状态运行。

## 🚀 快速启动

得益于 `Gradle`，只要您的环境中有 JDK 25 或更高版本，即可一件启动，不需预先安装其他依赖。

### 1. 本地直接运行

```bash
# Windows
./gradlew.bat run

# Linux / macOS
./gradlew run
```

### 2. 编译并打包单文件 JAR

如果需要将它部署到服务器上，您可以编译为一个包含了所有依赖资源的胖可执行 Jar（Fat-Jar），一键随处运行。

```bash
# 生成单文件可执行 Jar
./gradlew shadowJar

# 运行（通过 nohup, systemd 等工具挂载即可）
java -jar build/libs/fsh-1.0-SNAPSHOT-all.jar
```
*(注意：请确保构建配置里已启用 shadow 插件进行胖打包)*

第一次运行时，会在当前目录自动生成 `config.json` 配置文件。您可以自行关闭后修改参数后再运行。

## ⚙️ 配置说明

服务在启动时会自动于同级目录生成/读取 `config.json` 文件：

```json
{
  "port" : 8080,
  "storagePath" : "./data/fsh_uploads",
  "maxRequestSize" : 2147483648,
  "adminPathPrefix": "/manage" // <- 此为管理后台的前缀
}
```

- `port`: 应用启动绑定的 HTTP 端口（默认：8080）。
- `storagePath`: 上传文件的存放目录。所有的数据文件和 json 描述均会扁平存放于此（默认：`./data/fsh_uploads`）。
- `maxRequestSize`: 单个请求支持的最大字节数，默认为 2GB (2147483648 字节)。如需上传超大文件，可提升该值。
- `adminPathPrefix`: 访问管理控制台页面的 Base URL 前缀，避免被外界轻易扫描利用（留空 `""` 则在根目录 `/` 展示）。不影响 `http://localhost:8080/f/...` 的单纯下载链接。

---

如果您的配置前缀是 `/manage`，您可以现在访问: `http://localhost:8080/manage/` 体验 fsh。
